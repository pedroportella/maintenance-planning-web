# UI Assets Package

Boundary for local images, icons, wordmark metadata and other static assets owned by the web repo.

Assets should support the planner workbench experience and remain synthetic or generic.

## Current Assets

- `workbenchBrand` provides neutral wordmark metadata for the app shell.
- `workbenchIconNames` maps generic Lucide icon names to planner routes and shell actions.
- `assetProvenanceNotes` records where the current asset metadata came from.

Do not add employer, client, vendor or real source-system branding to this package.
